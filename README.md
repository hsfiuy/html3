# Quik Stop Subs Website

A modern website for Quik Stop Subs with smooth transitions and multiple pages.

## Pages
- **Home (index.html)**: Slideshow of food, descriptions, reviews, videos.
- **Food Showcase (food.html)**: Gallery of food images with captions and menu.
- **Place Showcase (place.html)**: Gallery of place images.
- **Contact (contact.html)**: Contact information with special background.

## Setup
1. Place images in `images/` folder: f1.png to f9.png, home1.png, home2.png, home3.png, menu1.png.
2. Place videos in `videos/` folder: video1.mp4, video2.mp4, etc.
3. Open index.html in a browser to view the site.

## Features
- Responsive design with CSS Grid and Flexbox.
- Smooth slideshow with fading transitions.
- Color palette: #B59077, #ACE6AE, #705D53, #F07865, #FFF7AD.

## Troubleshooting
- If images/videos don't load, ensure files are in the correct folders.
- For blurred background on contact page, ensure home3.png is present.
